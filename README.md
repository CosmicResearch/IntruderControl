# IntruderControl
Repo for the Mission Control software used by Cosmic Research in its Ansari (Intruder) launchment.
Includes the software for a Raspberrypi + Adafruit Ultimate GPS + PiCamera and for an Arduino + Adafruit 10-DOF IMU + Adafruit SD reader.
The raspberrypi code has been succesfully tested with a Raspberrypi zero at the Cosmic Research Ansari's launch (05/08/2017).
